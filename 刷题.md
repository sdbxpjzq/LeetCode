## _112_路径总和_简单

```
给定如下二叉树，以及 目标和sum = 22，

              5
             / \
            4   8
           /   / \
          11  13  4
         /  \      \
        7    2      1
返回 true, 因为存在目标和为 22 的根节点到叶子节点的路径 5->4->11->2。

```

## _226_翻转二叉树_简单

```
输入：

     4
   /   \
  2     7
 / \   / \
1   3 6   9
输出：

     4
   /   \
  7     2
 / \   / \
9   6 3   1
```
根节点的左右子节点 互换

```

   1
 /   \

解释: 所有根节点到叶子节点的路径为: 1->2->5, 1->3

```
